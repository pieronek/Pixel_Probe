
# Contributing

Thanks for considering contributing!

## Environment
- Python 3.8+
- Create venv:
  ```bash
  python -m venv .venv
  . .venv/Scripts/activate  # Windows PowerShell: .venv\Scripts\Activate.ps1
  pip install -U pip
  ```

## Workflow
- Format: `black .` (if used)
- Lint: `ruff check .` (if used)
- Test: `pytest`

## Pull Requests
- Keep PRs small and focused.
- Include before/after screenshots if changing UI.
- Link to related issues.
